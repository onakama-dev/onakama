Step① Supabase CLI インストール（まだなら）

ターミナルで：

npm install -g supabase


確認：

supabase --version

Step② プロジェクトと接続

SupabaseのWeb画面で：

Settings
→ General
→ Project ID をコピー

ターミナルで：

supabase login


ブラウザが開く → 認証

次に：

supabase link --project-ref あなたのProjectID

Step③ Edge Function 作成
supabase functions create send-email


すると

supabase/functions/send-email/index.ts


が作られます。

Step④ コードを編集

index.ts を開いて、全部これに置き換え：

import { serve } from "https://deno.land/std@0.168.0/http/server.ts"
import { Resend } from "npm:resend"

const resend = new Resend(Deno.env.get("RESEND_API_KEY"))

serve(async (req) => {
  const { to, subject, html } = await req.json()

  const data = await resend.emails.send({
    from: "onboarding@resend.dev",
    to,
    subject,
    html,
  })

  return new Response(JSON.stringify(data), {
    headers: { "Content-Type": "application/json" },
  })
})

Step⑤ APIキーをSupabaseに登録

これ超重要です。

ターミナルで：

supabase secrets set RESEND_API_KEY=ここにAPIキー


例：

supabase secrets set RESEND_API_KEY=re_xxxxxxxxxxxxx

Step⑥ デプロイ
supabase functions deploy send-email


成功するとURLが表示されます。

Step⑦ 呼び出し側（チャット投稿時）

チャット送信成功後に：

await fetch(
  "https://あなたのprojectid.supabase.co/functions/v1/send-email",
  {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
    },
    body: JSON.stringify({
      to: "通知先メールアドレス",
      subject: "新しいチャットが届きました",
      html: "<p>新しいメッセージがあります</p>",
    }),
  }
)

🎯 補足（本番で重要）
通知先メールはどう決める？

チャット相手のemailを：

auth.users


から取得できます。
